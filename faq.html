How do I install Taco?

If you have encountered or used YouTube “tutorials” on how to install and use Taco, please disregard them.
Many of them are incorrect or outdated and, as stated below, we do not support older versions.


To install Taco Client:

  1. Download the Taco Client, choosing one of two options:
    • Release: this version is updated less frequently, but is preferred if you don’t want to update often.
      You can download it on the official website;
    • Create a Fabric instance in your Minecraft launcher, and choose the latest version of Minecraft;
    • Go in the mods folder of the newly created instance, and paste the Taco-client-x.x.x-xxxx.jar file in it;
    • Run the instance, and if you did everything correctly, you should see a text in the top right corner indicating that you are using Taco.

How do I open the GUI?

The default key for opening the GUI is Right Shift, which you can change this via the Minecraft controls menu.
Inside the GUI, you can left-click on a Module to toggle it and right click to configure its settings.

How do I use the HUD?

  • Go to the HUD tab (the top of the screen).
  • Check the “Active” checkbox to enable it.
  • Tweak any settings you would like and use the “Edit” button to access the HUD editor.

A red highlight on the hud element indicates that they are OFF. Simply left click to toggle and right click to configure.

How do I import and export configs?

You can use Copy and Paste shortcuts in the GUI to copy and paste configs.

  • Windows and Linux: Ctrl + C and Ctrl + V
  • MacOS: ⌘ + C and ⌘ + V

You can also use this feature in individual modules and individual tabs (GUI, Config, HUD, etc).

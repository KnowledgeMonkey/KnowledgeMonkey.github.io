Installation Guide

To install Taco Client, follow these steps:

  1. Ensure you have the latest version of Fabric API installed.
  2. Ensure you have the latest version of Fabric Mod Loader installed.
  3. Download the Taco Client .jar file (coming soon).
  4. Place the downloaded .jar file into the mods folder of your Minecraft directory.
  5. Launch Minecraft with the Fabric profile to start using Taco Client.
